About this project

1. The job of this project is to colorize Greyscale images. For this the image will be converted to greyscale and resized to 256*256 dimensions in order to feed in to the neural network

2. For this project we have used a Deep Convolutional Generative Adversarial Networks(DCGAN) and Patch discriminator

3. This neural nework is trained on around 50K images on COCO unlabeled 2017 dataset

4. Try to feed/upload more general scenes with sky and trees for better results
